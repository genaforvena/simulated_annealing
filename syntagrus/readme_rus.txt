В настоящем пакете находятся следующие материалы:

1) Корпус текстов, состоящий из 14 подкаталогов (2003-2014, News, Uppsala) каталога SynTagRus2014.

2) Каталог Structure Editor с программой Structure Editor (структурный редактор). Это облегченная версия большой многофункциональной программы, используемой для создания корпуса. Она обладает ограниченной функциональностью и предназначена для удобной визуализации размеченных текстов корпуса. Структурный редактор - исполняемая программа StrEd.exe, для работы которой нужны все находящиеся в каталоге файлы (динамические библиотеки и конфигурационный файл). 

3) Описание программы Structure Editor - документ StructureEditor.doc; описание новых возможностей Structure Editor - StructureEditorInstructions.docx.

4) Лингвистическое описание корпуса - документ Pamjatka2012.doc

5) Текстовый файл dict.txt, содержащий материал из зоны комментариев статей комбинаторного словаря. Такая зона в основном используется для того, чтобы различить многозначные слова (ср., например, комментарии и примеры к глаголам ТОЛКОВАТЬ1 (Истолковывать, интерпретировать) и ТОЛКОВАТЬ2 (беседовать). Увидеть имена этих многозначных слов в корпусе можно, заглянув в поле KSNAME диалогового окна WordProperties). Копия этого файла содержится также в каталоге Structure Editor. 
6) Лингвистическое описание аппарата лексических функций - документ LF_guidelines.doс

7) Подробный список примеров лексических функций и их значений - документ LF_mass_examples.doс
